Esta é a página inicial do site Ecommerce LOWCOST

A página Low Cost é uma tentativa de criação de página que tem como objetivo facilitar a busca por computadores para jogos ou para estudo, evitando que você precise pesquisar por computadores ou até mesmo tendo que ir ver reviews sobre o que você está procurando. Dentro do site você encontra as informações sobre configurações, preço e até mesmo review de pessoas que já compraram e estão mostrando.

Meu Projeto

O meu projeto se consiste em uma criação de uma pagina com o objetivo de ser um intermediador entre encontrar computadores, periféricos relacionados a computador, de uma forma mais "barata", e apresentar para as pessoas que estão em busca de encontrar uma forma mais rapida, sem que fique pesquisando na internet, é mais ou menos buscar os melhores computadores custo beneficio, obtendo assim também uma forma de conseguir ganhar um lucro em cima de cada compra que foi feita usando os links de navegação do site.

Por ter feito o projeto solo, tive muita dificuldade de Desenvolver, tive dificuldadede de escolher e até mesmo elaborar o projeto sozinho, alterei diversas vezes a ideia de querer desenvolver algo mais complexo pois sabia que não daria tempo, o projeto não ficou exatamete como gostaria pois tive pouco tempo, porém está como planejado.
Foi dificil fazer a parte responsiva mas deu certo.
